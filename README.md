KIWI 智能排版系统 Pro

KIWI 智能排版系统 Pro 是一个纯前端、离线可用的公文排版与格式修复工具。

在线使用

点击这里直接使用

主要功能

Word 版本上传与修复：支持上传 .docx 文件，自动识别并修复格式。

表格完美还原：深度解析 Word 表格，1:1 还原合并单元格、边框和样式。

智能文本清洗：自动去除 AI 生成痕迹（Markdown 符号、废话前言），修正中英文标点。

公文规范排版：自动识别标题、章节、落款，应用标准的字体（仿宋/黑体/楷体）、字号和行距。

安全无忧：纯前端运行，所有文件处理均在本地浏览器完成，不上传服务器，保障数据安全。

使用方法

Word 修复：点击右上角“Word版本上传”，选择包含表格或文本的文档，系统将自动重构。

文本输入：直接粘贴 AI 生成的文本，点击“一键智能排版”。

预览与下载：在右侧预览效果，确认无误后点击“下载 Word”生成标准化文档。

技术栈

React 18

Tailwind CSS

Mammoth.js (Word 读取)

Docx.js (Word 生成)

Babel (JSX 解析)

本项目为静态单页应用，无需后端服务。
